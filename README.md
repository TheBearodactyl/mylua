# My Edits

Comments are now defined using `->`

```lua
-- Before
-> Now
```

`~=` is now `!=`

```lua
-- Before
if "aaaaa" ~= "bbbbb" then ... end

-> Now
if "aaaaa" != "bbbbb" then ... end
```
